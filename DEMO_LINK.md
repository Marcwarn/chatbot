# 🔗 Testa Big Five Demon Online

Eftersom localhost inte fungerar i din miljö, här är alternativ:

## Metod 1: CodePen (Enklast)

1. **Gå till:** https://codepen.io/pen/
2. Klistra in denna kod i HTML-fältet:

```html
<!DOCTYPE html>
<html lang="sv">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Persona - Big Five Test</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <script src="https://unpkg.com/lucide@latest"></script>
</head>
<body>
    <!-- Kopiera innehållet från big-five-demo.html -->
</body>
</html>
```

3. Klicka på filen `big-five-demo.html` i din Finder
4. Öppna med TextEdit
5. Kopiera ALLT innehåll
6. Klistra in i CodePen HTML-fältet
7. Klicka "Run"

## Metod 2: Ladda upp till Netlify Drop (Superenkelt!)

1. **Gå till:** https://app.netlify.com/drop
2. Dra och släpp filen `big-five-demo.html` från Finder
3. Du får en live URL direkt!

## Metod 3: Använd mitt GitHub Repo

Filen finns på GitHub:
https://github.com/Marcwarn/chatbot/blob/claude/personality-assessment-app-01DPbLhQT3Y5doG4v5gBJpXf/big-five-demo.html

Klicka "Raw" för att se den rena HTML-koden.

## Metod 4: Lokal fil (Mac Finder)

1. Öppna Finder
2. Gå till: `/home/user/chatbot/`
3. Hitta `big-five-demo.html`
4. Högerklicka → Öppna med → Safari
5. Eller dra filen till Safari/Chrome ikonens i Dock
